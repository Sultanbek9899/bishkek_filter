-Установите Python 3.8

-Создайте виртуальное окружение и загрузите зависимости из requirements.txt 
 pip3 install -r requirements.txt 

-В терминале запустите команду для создание базы данных
    1)pythnon3 manage.py makemigrations
    2)python3 manage.py migrate

-Создайте супер пользователя
    -python3 manage.py createsuperuser
        можете дать username:admin password: 123
  
-Запустите сервер: 
    -python3 manage.py runserver 

после этого заходите по ссылке:
        http://127.0.0.1:8000/

Но что-бы добавить товар: 
    надо перейти по ссылке 
        http://127.0.0.1:8000/admin
    и вести учетные данные,которые вы создовали до этого
    

**ВАЖНО
    У каждого товара есть пункты для акций ,  если вы поставите голачку в пункте    "Статус Акции" в панели , то ваш продукт будет уже показываться как продукт с акцией.Если вы сделали продукт акционным, то не забудьте добавить новую цену и размер скидки.
** Если хотите исключить данный продукт из акций , то просто уберите галочку в Пункт Статус Акции
